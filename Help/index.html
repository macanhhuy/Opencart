R.Gen Theme Features

Important! Information about theme layout

  • RGen theme is two columns only, it does not support 3 column layout.
  • You can add modules (Bestsellers, Latest, Featured, Specials, etc.) in either right column or left column do not set modules in both columns otherwise theme layout distorted


 

After modifying theme you just need to modify theme icon colours to match your theme which you can modify very easily with the file sprites.psd which is included in PSD file source

After modifying icons export file with 24 bit PNG format into following path

Take backup of original file before overwrite the file & do not modify positions of the icons in file otherwise it will not display properly in theme because all positions are set in theme.css file

upload/catalog/view/theme/rgen-opencart/image/rgen/

Overwrite the file when it ask for overwrite at the time of exporting

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

CSS Structure

Here is the list of css files that RGen OpenCart theme uses:

  • stylesheet.css – Main core stylesheet where whole structure is defined 
    File path => catalog/view/theme/rgen-opencart/stylesheet/stylesheet.css
  • carousel.css – Manufacturer logo slider stylesheet
    File path => catalog/view/theme/rgen-opencart/stylesheet/carousel.css
  • cloud-zoom.css – Image zoom stylesheet which used in product details page main image zoom functionality
    File path => catalog/view/theme/rgen-opencart/stylesheet/cloud-zoom.css
  • fb.css – Facebook fan list display stylesheet in custom footer at bottom the theme
    File path => catalog/view/theme/rgen-opencart/stylesheet/fb.css
  • scrollpane.css – Custom scroll bar stylesheet which is used in product details page
    File path => catalog/view/theme/rgen-opencart/stylesheet/scrollpane.css
  • nivo-slider.css – Slide show Stylesheet
    File path => catalog/view/theme/rgen-opencart/stylesheet/nivo-slider.css
  • store_home.css – Home page custom structure stylesheet
    File path => catalog/view/theme/rgen-opencart/stylesheet/store_home.css
  • theme.css – Main theme stylesheet where defined whole theme colours, icons, backgrounds images & other formatting you need to modify only this stylesheet if wish to make advance change in theme except from given in RGen Theme module
    File path => catalog/view/theme/rgen-opencart/stylesheet/store_home.css

The best practice for css customizations is to always put new styles at the bottom of the main stylesheet. This will help you to easily transfer all the changes, when you upgrade Theme.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

JavaScript

This theme imports below javascripts

  • common.js – The file contains all custom theme scripts and is not recommended to be modified.
  • jquery.carouFredSel-6.1.0-packed.js – This file use in general sliders in theme
  • jquery.touchSwipe.min.js – Use for touch event support
  • jquery.jscrollpane.min.js - This file use in product details page to set custom scroll bars
  • jquery.mousewheel.js – This file use in product details page to set mouse wheel event on scroll bars
  • cloud-zoom.1.0.2.min.js – This file use in product details page to magnifying product image on mouse roll over
  • jquery.nivo.slider.pack.js – This file use in slideshow

There are few other javascript files as well which are default included by OpenCart source you can find all javascript from below path

  • Default javascripts => upload/catalog/view/javascript/
  • R.Gen theme javascripts => upload/catalog/view/theme/rgen-opencart/js/

We strongly recommended do not modify any script
